## 如何贡献

尽可能保证引用资料的可信度和客观性。  
尽可能隐藏自己的真实身份，不要使用包含自己真实信息的GitHub账号提交Issue和PR，做好最坏的打算。  
如果不想通过GitHub在线的方式贡献，可以发送相关内容到evil-farben@proton.me。
